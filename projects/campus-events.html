project · mobile app

Campus Events

One feed for every symposium, hackathon and workshop a college runs. Students browse, follow and enroll. Organizers post an event in a few taps. I designed the screens and built the app, front and back.

Role: full stack Flutter Maps API Photo galleries Light + dark

overview

What it does

A college drops its events into the app and students find them all in one place. You can filter by All, Following or Trending, see who else is going, open an event for the full brief with poster and photos, and enroll in a tap. Organizers get a create flow that pins the venue on a map, invites friends, and attaches images before publishing the poster.

features

The parts that matter

Event feed

All, Following and Trending tabs, with participant avatars so you can see who is going.

Full event page

Time, fee, location, organizer, poster and a photo carousel, then a clear Enroll action.

Create an event

College, event, date and time, address with a Google Maps picker, and friend invites.

Light and dark

The whole app is themed both ways, tuned so text stays readable in either mode.
